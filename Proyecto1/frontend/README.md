# Frontend

Dashboard web en React + Vite + Tailwind para monitoreo y control del invernadero.

## Variables de entorno

- `VITE_API_BASE_URL`: URL base del backend, por ejemplo `http://localhost:8000`.

## Funciones principales

- muestra el estado general del sistema,
- consulta lecturas, eventos, comandos y logs recientes,
- envía acciones manuales al backend,
- registra lecturas y eventos desde la interfaz.

## Arranque

```bash
cd Proyecto1/frontend
pnpm install
pnpm dev
```
